h1. ring-basic-authentication

Ring middleware to enforce basic authentication as described in RFC2617 section 2.


h2. Usage

<pre>
(use 'ring.middleware.basic-authentication)

(defn authenticated? [name pass] (and (= name "foo") (= pass "bar")))

(def app (-> routes .. (wrap-basic-authentication authenticated?))
</pre>

if you use noir:
<pre>
(ns 'myapp.server
 (:use 'ring.middleware.basic-authentication)
 (:require [noir.server :as server]))

(server/add-middleware wrap-basic-authentication authenticated?)
</pre>

h2. Installation

Drop the following dependency in your <code>project.clj</code> at the appropriate place:

<pre>
[ring-basic-authentication "1.0.0"]
</pre>
